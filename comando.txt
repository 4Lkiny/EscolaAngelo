# Como CLONAR um repositorio do GOTHUB como visual code

1 No GOTHUB abre o repositório e clique no botão CODE -> Local -> copiar LINK do repositótio 
2 No visual code abra a pasta de seus projeto (exemplo: Documento/Python/)
3 Abra o terminal e digite o comando: git clone <cole aqui o link copiado
4 Tecle ENTER
5 No VS code abra a pasta clonado
6 No terminal digite git config user.name.<digite aqui seu nome de usuario. no Github>
7 tecle ENTER
8 Ainda no terminal digite: git config user.email <digite aqui seu Github>
9 tecle ENTER
